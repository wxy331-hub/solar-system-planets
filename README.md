# 探索太阳系：八大行星的奥秘

一个面向中学生、天文爱好者和大众的太阳系八大行星科普静态网站。

## GitHub Pages 发布

1. 在 GitHub 创建公开仓库：`solar-system-planets`。
2. 上传本目录全部文件和 `assets` 文件夹。
3. 进入仓库 `Settings` -> `Pages`。
4. Source 选择 `Deploy from a branch`。
5. Branch 选择 `main`，目录选择 `/root`。
6. 保存后等待 1-3 分钟。

发布地址通常为：

```text
https://wxy331-hub.github.io/solar-system-planets/
```
